# 钢管原料进出入库管理系统

## 项目简介

这是一个专为Windows用户设计的钢管原料进出入库管理系统，界面清晰直观，操作简便。系统采用Python和Tkinter开发，使用SQLite数据库存储数据，无需额外安装数据库软件。

## 功能特点

1. **钢管入库**：记录新钢管的入库信息，包括钢管编号、规格、材质、尺寸、数量、存放位置、供应商等详细信息。

2. **钢管出库**：记录钢管的出库信息，系统会自动检查库存数量，防止超量出库。

3. **库存查询**：查看当前库存的所有钢管信息，支持按关键字搜索。

4. **出入库记录**：查看所有出入库操作记录，支持按操作类型和钢管编号筛选。

5. **数据统计**：查看库存统计数据，包括总钢管种类数、总钢管数量、入库总数和出库总数。

## 系统要求

- Python 3.6 或更高版本
- Windows 操作系统

## 安装步骤

1. 确保已安装Python 3.6或更高版本
2. 下载本项目文件
3. 无需安装额外依赖，系统使用Python标准库

## 使用方法

1. 双击运行`main.py`文件
2. 系统会自动创建数据库文件`pipes.db`
3. 使用左侧导航栏切换不同功能

## 功能说明

### 钢管入库

- 填写钢管编号、规格、材质、尺寸、数量等信息
- 可选填写存放位置和供应商信息
- 必须填写操作员信息
- 点击"确认入库"按钮完成入库操作

### 钢管出库

- 输入钢管编号和出库数量
- 必须填写操作员信息
- 系统会自动检查库存数量
- 点击"确认出库"按钮完成出库操作

### 库存查询

- 查看当前库存的所有钢管信息
- 使用搜索框按关键字搜索
- 点击"刷新"按钮更新数据

### 出入库记录

- 查看所有出入库操作记录
- 可按操作类型（入库/出库）筛选
- 可按钢管编号筛选

### 数据统计

- 查看库存统计数据
- 包括总钢管种类数、总钢管数量、入库总数和出库总数

## 数据备份

系统数据存储在`pipes.db`文件中，定期备份此文件可防止数据丢失。

## 注意事项

1. 钢管编号必须唯一，重复的编号会自动累加数量
2. 出库数量不能超过当前库存数量
3. 操作员信息为必填项
4. 建议定期备份数据库文件
